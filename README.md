🌟 NemoWatch Pro - Ultimate Smartwatch for M5StickC Plus 🌟

Полноценные смарт-часы с современным интерфейсом и богатым функционалом!

⚡ ОСНОВНЫЕ ВОЗМОЖНОСТИ:
✅ Часы с погодой (автообновление через WiFi)
✅ Секундомер с миллисекундами
✅ Таймер с звуковым сигналом
✅ Будильник с настройкой времени
✅ Калькулятор
✅ Игра "Змейка"
✅ Фонарик
✅ Веб-настройка через браузер
✅ Красивый интерфейс в стиле Apple Watch

🎮 УПРАВЛЕНИЕ:
- Кнопка M5 (спереди) - выбор/подтверждение
- Кнопка Bok (сбоку) - навигация
- Держать M5+Bok 2 сек - открыть меню

⚙️ УСТАНОВКА:
1. Залейте прошивку через Arduino IDE
2. При первом запуске выберите WiFi сеть
3. Введите пароль через умную клавиатуру
4. Готово!

🔧 ТЕХНИЧЕСКИЕ ДЕТАЛИ:
- Платформа: M5StickC Plus (ESP32-PICO)
- Библиотеки: M5StickCPlus, WiFi, HTTPClient, WebServer
- Язык: C++ (Arduino)
- Яркость экрана: 80% (регулируется)
- Авто-сон через 30 сек бездействия

🌐 ВЕБ-ИНТЕРФЕЙС:
Откройте IP адрес часов в браузере для настройки:
- WiFi сеть и пароль
- Город для погоды
- Часовой пояс
- Будильник
